1. Sign Up Endpoint

Request Fields

  • name - User's full name
  • email - User's email address
  • phone_number - User's phone number
  • password - User's password
  • role - Either 'farmer' or 'grocer'
  • store_name - Required only if role is 'grocer'

2. Login Endpoint

Request Fields

  • identifier - Either email or phone number
  • password - User's password

3. Products Endpoint

GET Method

Returns all products

Response Format:

  • name
  • description
  • quantity
  • price
  • farmer

POST Method

Creates a new product (Farmer only)

Request Fields:

  • name - Product name
  • description - Product description
  • quantity_available - Integer value
  • price_per_unit - Integer value

4. Orders Endpoint

GET Method

Returns orders based on userId

For Farmers Response:

  • order_id
  • grocer_name
  • products[]
  • total_amount
  • order_date

For Grocers Response:

  • order_id
  • products[]
  • total_amount
  • order_date
